# Here is some general info about the website

## Used stack:
 - React
 - Typescript
 - ESlint *(for code rules and style)*
 - Sass
 - firebase *(for db connection)*

## Some useful scripts:

To start the website locally:

    npm start

To build for production:

    npm build
